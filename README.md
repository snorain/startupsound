# startupsound
This plays the QuintOS startup ring tone on startup.

# How to install
Clone the git repo:
```sh
git clone https://github.com/QuintOS/startupsound
```
cd into the main folder
```
cd satrtupsound
```
Run the install script
```
bash install
```

# How to uninstall:
cd into the main folder
```
cd satrtupsound
```
Run the uninstall script
```
bash uninstall
```
