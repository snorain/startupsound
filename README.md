# startupsound
This plays the snorain startup ring tone on startup.

# How to install
Clone the git repo:
```sh
git clone https://github.com/snorain/startupsound
```
cd into the main folder
```
cd satrtupsound
```
Run the install script
```
bash install
```

# How to uninstall:
cd into the main folder
```
cd satrtupsound
```
Run the yninstall script
```
bash yninstall
```
